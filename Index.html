02

Step-by-Step ইনস্টল গাইডStep-by-Step Install Guide

প্রতিটি step ক্লিক করে বিস্তারিত দেখুনClick each step to expand details


01
ফাইল প্রস্তুত করুনPrepare Files
ROM zip ও TWRP img সঠিক জায়গায় রাখুনPlace ROM zip and TWRP img in correct locations
▾
  • Memory Card / OTG Storage তে ROM zip কপি করুন → romfile.zipCopy ROM zip to Memory Card / OTG Storage → romfile.zip
  • TWRP img ফাইল Mother Phone এর internal storage তে রাখুন → twrp_recovery.imgPlace TWRP img in Mother Phone's internal storage → twrp_recovery.img
  • Mother Phone এ Bugjaeger Mobile ADB install করুনInstall Bugjaeger Mobile ADB on Mother Phone
ℹ️ ROM zip অবশ্যই Sunstone (Redmi Note 12 5G) এর জন্য হতে হবে। The ROM zip must be specifically built for Sunstone (Redmi Note 12 5G).
02
Fastboot Mode এ যানEnter Fastboot Mode
Volume Down + Power বাটন একসাথে চাপুনHold Volume Down + Power simultaneously
▾
  • Target Phone সম্পূর্ণ বন্ধ করুনCompletely power off the Target Phone
  • Volume Down + Power বাটন একসাথে চেপে ধরুনHold Volume Down + Power buttons simultaneously
  • Fastboot screen আসলে বাটন ছেড়ে দিনRelease when the Fastboot screen appears
  • OTG Cable দিয়ে Mother Phone এর সাথে কানেক্ট করুনConnect to Mother Phone via OTG Cable
  • Bugjaeger open করুন → "Device detected" দেখালে সফল ✅Open Bugjaeger → "Device detected" means success ✅
⚠️ Device না দেখালে cable খুলে আবার লাগান। অন্য OTG cable try করুন। If device not detected — unplug and replug. Try a different OTG cable.
03
Active Slot চেক করুন ⚡Check Active Slot ⚡
সবচেয়ে গুরুত্বপূর্ণ — এড়িয়ে যাবেন নাMost critical step — do not skip
▾
🔴 Slot ভুল হলে TWRP flash হবে না অথবা boot loop হবে। Wrong slot will prevent TWRP from flashing or cause a boot loop.

Bugjaeger → Fastboot tab → command দিন: Bugjaeger → Fastboot tab → run command:

fastboot getvar current-slot
Slot A
✓ Step 4 এ যানGo to Step 4

এটা দেখালে ঠিক আছেThis is correct — proceed

Slot B
⚠ পরিবর্তন করুনChange required

নিচের command দিনRun command below

Slot B থাকলে:If Slot B, run:

fastboot set_active a
04
TWRP Flash করুনFlash TWRP
Bugjaeger দিয়ে twrp_recovery.img flash করুনFlash twrp_recovery.img via Bugjaeger
▾
  • Bugjaeger → Fastboot tab → command box এ লিখুন:Bugjaeger → Fastboot tab → type in command box:
fastboot flash boot
  • twrp_recovery.img ফাইলটি select / drag করুন এবং Run করুনSelect / drag twrp_recovery.img and click Run

সফল হলে দেখাবে:On success you will see:

OKAY [ 0.xxx s] Finished. Total time: x.xxxs
⚠️ Error আসলে cable বদলান। Slot A active আছে কিনা নিশ্চিত করুন। On error — change cable. Make sure Slot A is active.
05
TWRP তে Boot করুনBoot into TWRP
fastboot reboot recovery দিনRun fastboot reboot recovery
▾
fastboot reboot recovery
ℹ️ অথবা Volume বাটন দিয়ে Recovery সিলেক্ট করে Power চাপুন। Or use Volume buttons to select Recovery, then press Power.
✅ TWRP screen আসলে সফল! TWRP screen means success!
⚠️ System boot হলে — Step 3 থেকে আবার করুন। If system boots instead — go back to Step 3.
06
Advanced Wipe করুনAdvanced Wipe
System সহ সব partition wipe করুনWipe all partitions including System
▾
  • TWRP → Wipe → Advanced Wipe এ যানGo to TWRP → Wipe → Advanced Wipe
  • নিচের সবগুলো টিক দিন:Check all of the following:
✓ Dalvik / ART Cache
✓ System
✓ Data
✓ Metadata
✓ Cache
  • Swipe to Wipe করুন — 100% হওয়া পর্যন্ত অপেক্ষা করুন ⏳Swipe to Wipe — wait until 100% complete ⏳
⚠️ System টিক না দিলে পুরনো ROM conflict করে boot loop হতে পারে। Skipping System may cause old ROM conflicts and boot loop.
07
Slot চেক ও Recovery তে RebootCheck Slot & Reboot to Recovery
System এ reboot করবেন না এখনোDo NOT reboot to System yet
▾
  • TWRP → Reboot এ যানGo to TWRP → Reboot
  • Current Slot দেখুন — Slot B দেখালে Slot A তে switch করুনCheck current slot — if Slot B, switch to Slot A
  • Reboot to Recovery চাপুনTap Reboot to Recovery
🔴 System এ Reboot করবেন না — ROM এখনো install হয়নি। Do NOT reboot to System — ROM is not installed yet.
08
ROM Install করুনInstall ROM
Memory Card থেকে romfile.zip flash করুনFlash romfile.zip from Memory Card / OTG
▾
  • TWRP → Install বাটনে যানTap Install in TWRP
  • Memory Card / OTG থেকে romfile.zip সিলেক্ট করুনSelect romfile.zip from Memory Card / OTG
  • Swipe to Confirm Flash করুনSwipe to Confirm Flash
  • Progress bar শেষ হওয়া পর্যন্ত অপেক্ষা করুন ⏳Wait until the progress bar completes ⏳
⚠️ Flash এর সময় ফোন নাড়াবেন না, cable খুলবেন না। Do not move the phone or disconnect the cable during flashing.
09
Format Data করুন ⚡Format Data ⚡
এই step skip করলে boot loop হবে — আবশ্যকMandatory — skipping causes boot loop
▾
  • TWRP → Wipe → Format Data এ যানGo to TWRP → Wipe → Format Data
  • Keyboard এ yes টাইপ করুন এবং Confirm করুনType yes and confirm
🔴 এই step অবশ্যই করতে হবে। Skip করলে প্রায় নিশ্চিত boot loop হবে। This step is mandatory. Skipping it almost guarantees a boot loop.
10
Reboot System করুনReboot to System
১০ মিনিট পর্যন্ত ধৈর্য ধরুনBe patient — first boot takes up to 10 minutes
▾
  • TWRP → Reboot → System চাপুনTWRP → Reboot → tap System
  • প্রথম boot এ ৫-১০ মিনিট সময় লাগবে ⏳First boot takes 5–10 minutes ⏳
  • MIUI / HyperOS লোগো আসলে সফল 🎉MIUI / HyperOS logo appearing means success 🎉
⚠️ Logo দেখে ধৈর্য হারিয়ে বন্ধ করবেন না। Do not power off when you see the logo.
🎉 ফোন চালু হয়ে গেলে Custom ROM install সম্পন্ন! Once the phone fully boots, Custom ROM installation is complete!
11
Root করুন — MagiskRoot with Magisk
Optional — root access এর জন্যOptional — for root access
▾
  • Magisk APK ডাউনলোড ও install করুনDownload and install Magisk APK
  • Magisk open → Install → Direct Install (Recommended)Open Magisk → Install → Direct Install (Recommended)
  • Reboot করুন → Root হয়ে গেছে ✅Reboot → Device is now rooted ✅
ℹ️ Direct Install কাজ না করলে — boot.img patch করে Fastboot দিয়ে flash করুন। If Direct Install fails — patch boot.img and flash via Fastboot.
